What is Union-Find Problem

Connectivity

\(a-b\): \(a\) and \(b\) are connected

Properties:

  • \(a-a\)

  • \(a-b \Longrightarrow b-a\)

  • \(a-b,b-c \Longrightarrow a-c\)

Connected Component

A is a connected component: \(a\in{A},b\in{A} \iff a-b\)

Union-Find

\(union(a,b)\): connect \(a\) and \(b\)

\(find(a,b)\): check if \(a\) and \(b\) are connected, return True or False

Problem: \(Union\) any \(N\) objects \((N\to\infty)\), then \(\forall a,b,Find(a,b)\) (Union at any time, Find at any time)
